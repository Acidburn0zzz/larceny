Strings
~~~~~~~

proc:string-downcase[args="string",result="string"]

Returns a string that is the same as _string_ except that every letter
is lower case.

proc:string-downcase![args="string",result="string"]

Converts every upper case letter in _string_ to lower case and returns
_string_.

proc:substring-fill![args="string start end char",result="unspecified"]

Fill _string_ with _char_ from location _start_ through location
__end__-1.


proc:string-upcase[args="string",result="string"]

Returns a string that is the same as _string_ except that every letter
is upper case.

proc:string-upcase![args="string",result="string"]

Converts every lower case letter in _string_ to upper case and returns
_string_.

See also: <<string-hash>>.
